
<h1>3D Portfolio</h1>

<p>Welcome to my 3D portfolio repository!</p>

<h2>Installation</h2>
<p>To set up the project locally, follow these steps:</p>

Make sure you have [Node.js](https://nodejs.org/) installed.

<pre><code>git clone https://github.com/SyartDev/my-portfolio.git
</code></pre>

<pre><code>npm install
</code></pre>

<pre><code>npm run dev
</code></pre>
